# Calculadora em Python

Este é um projeto simples de calculadora de linha de comando desenvolvido em Python. O objetivo principal é praticar conceitos básicos da linguagem, como:

* Uso de funções
* Estruturas de repetição
* Condicionais
* Tratamento de exceções

## Funcionalidades

* Soma de dois números
* Subtração
* Multiplicação
* Divisão com tratamento para divisão por zero
* Interface interativa via terminal
* Validação de entradas inválidas

## Como executar

1. Certifique-se de ter o Python 3 instalado.
2. Clone este repositório ou copie o código para um arquivo chamado `calculadora.py`.
3. Execute o programa pelo terminal com o comando:

```bash
python calculadora.py
```

4. Siga as instruções exibidas para realizar operações ou digite "5" para sair.
